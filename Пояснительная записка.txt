Страницы сайта:
/ - главная страница для клиента (небольшая информация о Пиццмане и Калачеве)
/menu - страница меню для клиента. Здесь клиент может добавить товар в корзину, перейти на страницу товара
/product/<int: id> - страница продукта. Картинка, информация о продукте.
/basket - корщина клиента. Здесь клиент может изменить кол-во товаров и оформить заказ
/login - страница входа для клиента
/profile - страница профиля клиента. Здесь можно редактировать информацию, добавить адреса доставки, посмотреть исторю заказов
/cook/login - страница входа для повара
/cook/orders - страница заказов для повара
/deliveryman/login - страница входа для доставщика
/deliveryman/orders - страница заказов для доставщика
/admin/login - страница входа для администратора
/admin/ - главная страница для администратора с которой он может пкрейти на страницы регистрации поваров или доставщиков.


В функции main() в файле main.py закомменчены создания первых администратора, повара и доставщика.
